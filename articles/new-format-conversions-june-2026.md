I’ve just released a new MConverter update. Here is what’s new:

* Added support for [KEPUB files](https://mconverter.eu/convert/kepub/). This format is a specialized version of EPUB that’s used on Kobo eReaders. If you have a Kobo device, it’s better to convert any e-books you have from [EPUB to KEPUB](https://mconverter.eu/convert/epub/kepub/), as this will make use of additional performance optimizations in Kobo’s rendering software. Or, simply put, your books will load faster.  
* More Office formats can be converted to Markdown. More specifically:  
  * [Excel spreadsheets to Markdown](https://mconverter.eu/convert/xlsx/md/)  
  * [PowerPoint presentations to Markdown](https://mconverter.eu/convert/pptx/md/)  
* [HTML to TXT](https://mconverter.eu/convert/html/txt/) conversions are now supported.  
* Large file uploads should be more reliable, especially on very slow mobile networks.  
* Fixed a missing push notification when all file conversions finish on Chrome Beta, Dev, Canary.

If you notice any issues or if you’d like to convert to KEPUB from a format that’s not currently supported, please [let us know](#legal) and it’ll be added.
